# /kokoro-red-viva — Mapa de Ecosistema y Alianzas

> Dinamica Kokoro inspirada en facilitacion visual para entender compras
> alrededor, relaciones, alianzas y contexto de uso.

## Contexto

Usa Red Viva cuando la creacion no vive sola: depende de personas, roles,
lugares, herramientas, servicios, proveedores, momentos o compras que ocurren
alrededor. Sirve para encontrar alianzas, bundles, integraciones y nuevas
oportunidades de posicionamiento.

Lee `kokoro-dinamicas-vivas.md` antes de iniciar.

## Antes de Comenzar

Pide permiso:

> "Me gustaria mirar tu creacion como si estuviera en el centro de una red.
> No vamos a forzar respuestas; vamos a descubrir que vive alrededor. Me das
> permiso de guiarte?"

## Preparacion

Define el centro de la red:

- Una creacion.
- Un buyer persona.
- Una situacion de compra.
- Un momento de uso.
- Un mercado local.

Elige de dos a cuatro relaciones a explorar:

| Relacion | Pregunta |
| --- | --- |
| Corporativa | Que marcas, aliados, proveedores o instituciones rodean esta compra? |
| Lugar / entorno | Donde ocurre, antes y despues de que momento? |
| Operativa | Que herramientas o servicios necesita para completar la tarea? |
| Humana | Quien influye, usa, recomienda, bloquea o acompana? |
| Rol | Que roles participan y que responsabilidad tiene cada uno? |

## Juego

1. Dibuja el centro.
2. Pide a la persona que agregue entidades alrededor.
3. Une entidades con lineas.
4. Usa grosor, color o etiquetas para mostrar fuerza, frecuencia, friccion o
   valor.
5. Pregunta "cuando, como y por que aparece esta relacion?"
6. No limpies demasiado el mapa. La red real suele ser desordenada.

## Preguntas Poderosas

- "Que compra esta persona antes de necesitarte?"
- "Que compra despues?"
- "Quien ya tiene su confianza?"
- "Que relacion te sorprende porque no la estabas considerando?"
- "Que alianza podria reducir friccion sin quitar paz mental?"
- "Que parte de esta red muestra una oportunidad de ingreso o referidos?"

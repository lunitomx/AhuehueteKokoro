# Funnel Consciente

> Knowledge file para `/kokoro-funnel`.

## Principio

Un funnel no es una tubería — es una conversación escalonada.
Cada etapa responde a una pregunta distinta del invitado:
1. ¿Existes? (awareness)
2. ¿Me entiendes? (consideración)
3. ¿Puedo confiar? (decisión)
4. ¿Valió la pena? (post-compra)

Kokoro audita cada etapa y recomienda la palanca de mayor impacto.

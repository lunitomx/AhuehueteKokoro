# Generación de Creativos con IA

> Knowledge file para `/kokoro-creative`.

## Proceso

1. **Brief visual** — qué comunicar, a quién, con qué emoción
2. **Generación** — Gemini API con prompt estructurado
3. **Revisión** — criterios Kokoro (no genéricos)
4. **Entrega** — creativo listo para Meta Ads

## Restricciones

- Nunca generar sin brief validado
- Nunca usar imágenes con texto ilegible
- Siempre 2+ variantes por concepto

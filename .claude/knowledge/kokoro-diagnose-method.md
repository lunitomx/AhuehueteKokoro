# Diagnose Method — Speed Boat + Vision 20/20

> Knowledge file para `/kokoro-diagnose` y sus sub-comandos.

## Speed Boat (causas raíz)

Metodología de diagnóstico que identifica qué está frenando el negocio:
- ¿Qué anclas arrastra el barco?
- Priorización por impacto y facilidad de resolución
- Output: Top 3 causas raíz con plan de acción

## Vision 20/20 (claridad estratégica)

Ejercicio de alineación: ¿dónde está el negocio hoy vs dónde debería estar?
- 20 preguntas de diagnóstico
- Visión a 3 años
- Gap analysis

## Sub-comandos

- `/kokoro-diagnose-report` — genera reporte ejecutivo
- `/kokoro-diagnose-speedboat` — solo Speed Boat
- `/kokoro-diagnose-vision2020` — solo Vision 20/20

Los tres heredan la metodología de este archivo.

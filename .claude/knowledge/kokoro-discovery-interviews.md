# Entrevistas de Validación

> Knowledge file para `/kokoro-interviews`.

## Metodología

Guía para entrevistar clientes potenciales sin venderles:
- **No pitches.** Escuchar, no hablar.
- **Preguntas de problema**, no de solución.
- **Evidencia de dolor**, no opiniones.
- **5-10 entrevistas** por segmento antes de decidir.

Kokoro ayuda a preparar el guión, procesar transcripciones y extraer
patrones validados.

# Scorecard Ejecutivo Cross-Platform

> Knowledge file para `/kokoro-scorecard`.

## Métricas consolidadas

- Meta Ads: inversión, conversiones, CPA, CTR
- Google Ads: inversión, conversiones, CPA, impresiones
- GA4: tráfico, conversiones, tasa de rebote
- Search Console: clics, impresiones, posición media

## Formato

Tabla semanal con variación vs semana anterior. Una sola vista.
Sin excusas, sin narrativa — solo números y flechas.

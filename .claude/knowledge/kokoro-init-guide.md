# Inicializar Kokoro en un Proyecto

> Knowledge file para `/kokoro-init`.

## Qué hace

Copia los archivos de conocimiento de Kokoro a un proyecto nuevo:
- `.claude/commands/` → slash commands
- `.claude/knowledge/` → archivos de referencia
- `.claude/skills/` → skills de calidad web
- `.gitignore` con reglas de privacidad

## Cuándo usarlo

- Proyecto greenfield que usará Kokoro
- Proyecto existente que quiere adoptar la metodología

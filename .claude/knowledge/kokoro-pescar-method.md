# Metodología PESCAR

> Knowledge file para `/kokoro-pescar`.

## Las 6 fases

1. **P**roblema — definir el dolor exacto
2. **E**videncia — buscar pruebas de que el dolor existe
3. **S**olución — diseñar la propuesta mínima
4. **C**anal — elegir dónde encontrarlos
5. **A**dquisición — diseñar el primer experimento
6. **R**esultado — medir y decidir siguiente paso

Kokoro guía cada fase con preguntas estructuradas y gates de validación.

# Podar el Árbol de Productos

> Knowledge file para `/kokoro-prune`.

## Metodología

El árbol de productos es todo lo que el negocio ofrece. Podar significa
eliminar líneas, servicios o creaciones que:
- No generan margen
- Drenan atención del core
- Confunden al mercado
- No tienen futuro estratégico

## Criterios de poda

1. **Rentabilidad:** ¿genera más de lo que cuesta?
2. **Alineación:** ¿sirve a la visión de 3 años?
3. **Energía:** ¿cuánta atención del fundador consume?
4. **Escala:** ¿puede crecer sin multiplicar complejidad?

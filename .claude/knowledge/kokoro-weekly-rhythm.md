# Ritmo Semanal + Scorecard

> Knowledge file para `/kokoro-rhythm`.

## Los 90 minutos semanales

1. **15 min** — Revisar scorecard (métricas que importan)
2. **30 min** — Analizar 1-2 palancas (no todo)
3. **30 min** — Decidir siguiente experimento
4. **15 min** — Documentar aprendizaje

## Scorecard

5-7 métricas máximo. Las que mueven el negocio, no las que son fáciles
de medir. Si una métrica no cambia una decisión, no va en el scorecard.

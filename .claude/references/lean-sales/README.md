# Lean Sales — Sistema Operativo de Ventas Inteligentes

> Creado por **Gerardo Osorio** · [Mhservices.mx](https://mhservices.mx)

## ¿Qué es Lean Sales?

Lean Sales es un sistema operativo para la función comercial que combina
metodologías ágiles (Lean Startup, Scrum, Kanban) con inteligencia artificial
para transformar la manera en que los equipos de ventas operan, aprenden y
entregan resultados.

No es un CRM ni un manual de ventas. Es un **sistema de gobernanza** que
estructura el caos comercial en procesos repetibles y medibles.

## Filosofía Lean Sales

1. **Gobernanza que no se siente como gobernanza** — disciplina que permite
   libertad, como un arte marcial donde las restricciones crean capacidad
2. **Jidoka** — parar en defectos, no acumular errores
3. **Tríada** — Humano (juicio) → Metodología (estructura) → IA (ejecución)
4. **Dogfooding** — construido usándose a sí mismo
5. **Inferencia económica** — recopilar con herramientas, pensar con inferencia

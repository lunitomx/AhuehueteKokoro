# Kokoro para Claude Desktop

> Kokoro se usa principalmente con Claude Code (CLI), no con Claude Desktop.
> Esta guía te ayuda a conectar skills clave vía MCP si usas Desktop.

## Ruta recomendada

Para la experiencia completa de Kokoro (68+ skills, 58 knowledge files, identidad
completa), usa **Claude Code**:

```bash
git clone https://github.com/lunitomx/AhuehueteKokoro.git
cd AhuehueteKokoro
claude
```

## Para Claude Desktop (opcional)

Si prefieres Claude Desktop, instala los servidores MCP para conectar tus plataformas:

- **Meta Ads**: sigue la guía en `/kokoro-mcp-reference`
- **Google Ads**: sigue la guía en `/kokoro-mcp-reference`
- **Google Analytics**: sigue la guía en `/kokoro-mcp-reference`
- **Google Search Console**: sigue la guía en `/kokoro-mcp-reference`

# S40.2 Story: Executive Router

## User Story

As a founder, marketing director, or sales leader who just installed Kokoro,
I want to describe what I need in business language,
so that Kokoro routes me to the right process without requiring me to know internal skill names.

## Acceptance Criteria

### AC1: Business Intent Router Exists

Given a user opens Kokoro without knowing which skill to run,
when they state an outcome such as "review acquisition" or "prepare my marketing week",
then Kokoro maps the intent to the appropriate executive workflow.

### AC2: Router Separates Strategic Phase Work From Operating Runs

Given a user needs foundational strategy,
when they describe uncertainty about diagnosis, vision, focus, model, validation, or launch,
then Kokoro routes to the existing phase skills.

Given a user needs an operational marketing decision,
when they ask about Google Ads, weekly marketing, launch execution, acquisition, or share readiness,
then Kokoro routes to the E40 orchestrator layer.

### AC3: Router Does Not Overpromise Missing Orchestrators

Given some E40 orchestrators are planned but not implemented yet,
when the router names them,
then it clearly labels planned flows as "planned" and points to the currently available underlying skills.

### AC4: Router Preserves Kokoro Voice And Privacy

Given the router asks clarifying questions,
when it needs context,
then it uses Kokoro vocabulary and never requests secrets, tokens, or public posting of guest data.

## Examples

### Executive Input

```text
Quiero saber como va mi marketing esta semana.
```

Expected route:

```text
Primary route: /kokoro-weekly-marketing-run (planned)
Available now: /kokoro-scorecard + /kokoro-analytics + /kokoro-pulse
First gate: resolve invitado and connected platforms.
```

### Strategic Input

```text
No se cual es el siguiente paso para mi negocio.
```

Expected route:

```text
Primary route: /kokoro-diagnose
Reason: first clarify the current terrain before choosing a growth process.
```

### Safety Input

```text
Quiero compartir Kokoro con un invitado.
```

Expected route:

```text
Primary route: /kokoro-share-readiness (planned)
Available now: .claude/knowledge/kokoro-share-readiness.md
First gate: no guest data, no credentials, no private runtime state.
```

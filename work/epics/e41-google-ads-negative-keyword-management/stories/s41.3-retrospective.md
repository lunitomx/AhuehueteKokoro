## Runtime path decision

No Python adapter code exists in this public repo (by design — no credentials,
env vars, or account mappings live here). The adapter executes as a Hermes
skill or local Python module in the operator's private runtime. The public
repo carries the contract and command surface only.

This matches the S41.2 contract decision: the official Google Ads MCP
server is read-only for negative keywords, so all mutation is local.

# S41.4 Scope: Audit + Docs + Regression Coverage

## Objective

Alinear auditoría, documentación y cobertura de regresión para la operación de eliminación de negativas.

## In Scope

- Documentar formato de auditoría equivalente a `add`.
- Publicar ejemplos de uso seguros.
- Definir regresión mínima para errores y éxito parcial.
- Verificar que `add` permanece sin cambios.

## Out of Scope

- Implementar la mutación de negativas desde cero.
- Cambiar el servidor oficial de Google Ads MCP.
- UI o reporting externo.

## Dependencies

- S41.1 discovery.
- S41.2 contract.
- S41.3 adapter path or explicit block.

## Done Criteria

- [ ] Auditoría documentada.
- [ ] Ejemplos publicables sin secretos.
- [ ] Casos de regresión definidos.
- [ ] `add` queda explícitamente protegido.

## Verification

- Revisar que los ejemplos usan placeholders.
- Revisar que la auditoría no expone datos sensibles.

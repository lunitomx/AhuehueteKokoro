# S42.1 Scope

## Objective
Make the learning record explicit enough to support Google Ads control and
kaizen, while preserving backward compatibility with current Kokoro sessions.

## In Scope
- Add Google Ads fields to the learning record model.
- Keep `summary`, `hallazgos`, `artifacts`, and `next_action` as core fields.
- Document how the schema maps to the learning dashboard.

## Out of Scope
- Any private persistence layer.
- Any MCP integration.
- Any dashboard UI.

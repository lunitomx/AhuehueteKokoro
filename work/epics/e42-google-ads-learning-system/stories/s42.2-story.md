# S42.2 Story: Persist Learning on Close

## Story
Update the close flow so every Google Ads session persists the learning record
with summary, hallazgos, next_action, and relevant campaign context.

## Outcome
Kokoro closeouts become durable learning entries instead of loose notes.

## Acceptance
- `/kokoro-close` writes the learning record with the dashboard vocabulary.
- `/kokoro-gads` can record Google Ads-specific context when applicable.
- The record remains compact and readable.
- No private runtime data enters the public repo.

## Dependencies
- S42.1
- `.claude/commands/kokoro-close.md`
- `.claude/commands/kokoro-gads.md`

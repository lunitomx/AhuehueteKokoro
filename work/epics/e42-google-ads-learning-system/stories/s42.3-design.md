# S42.3 Design

## Current Surface
- `kokoro-open` reads `metadata["session_log"]`.

## Target Behavior
- Add a concise view of recent learning entries.
- Keep the latest `next_action` visible.
- Show `learning_state` when present for Google Ads sessions.

## Presentation Rule
- Do not overload the open view with raw detail.
- Keep only the last few entries and the strongest next step.

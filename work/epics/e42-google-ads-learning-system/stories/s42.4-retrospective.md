# S42.4 Retrospective: Client Learning Guide

## What was implemented

Created `.claude/knowledge/kokoro-client-learning-guide.md` — a client-facing
guide that explains the Google Ads learning system in plain Spanish:

- What the Learning Dashboard is (Plan/Ejecución/Aprendizaje)
- What information Kokoro stores per session
- How to use the history across sessions
- What Kokoro does NOT store (credentials, billing, full conversations)
- Attribution to LeanStack and Ash Maurya

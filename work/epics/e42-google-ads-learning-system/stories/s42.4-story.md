# S42.4 Story: Client Learning Guide

## Story
Write the client-facing guide that explains how to read the learning dashboard
and how to use the plan / execution / learning loop.

## Outcome
Clients can understand the system without needing the internal authoring model.

## Acceptance
- The guide explains the three layers.
- The guide explains how LeanStack fits in.
- The guide keeps attribution to LeanStack / Ash Maurya explicit.
- The guide does not describe private runtime internals.

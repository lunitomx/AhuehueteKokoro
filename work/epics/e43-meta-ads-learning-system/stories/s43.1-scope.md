# S43.1 Scope: Meta Ads Learning Record Schema

## Objective

Define Meta Ads-specific fields for the Kokoro learning record, additive
to the S42.1 schema. Same vocabulary, same backward compatibility, new
platform-specific context.

## In Scope

- Add Meta Ads fields: `campaign_objective`, `audience_type`, `placements`, `creative_count`, `corpus_angle`
- Keep `platform`, `learning_state`, and all core fields shared with Google Ads
- Document the schema extension in the canonical learning record doc
- Ensure backward compatibility

## Out of Scope

- Creating a separate schema file
- Changing the existing Google Ads fields
- Any code or runtime

## Dependencies

- S42.1 Learning Record Schema (complete)

## Done Criteria

- [ ] Meta Ads fields added to `kokoro-google-ads-learning-record.md` (or renamed to platform-agnostic)
- [ ] All fields optional and backward compatible
- [ ] Schema documented with example entry

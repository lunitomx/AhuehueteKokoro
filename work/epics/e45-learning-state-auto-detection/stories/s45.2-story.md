# S45.2 Story: Google Ads Learning State Detector

## User Story
Como operador de Kokoro, quiero que el learning_state de Google Ads se infiera
automáticamente desde datos del MCP, con thresholds específicos por tipo de campaña.

## Acceptance Criteria
AC1: Detector implementado con decision tree documentado
AC2: Thresholds por campaign type (Search 14d/30conv, PMax 28d/50conv, etc.)
AC3: Fallback null cuando MCP no disponible
AC4: Override manual con prioridad sobre automático
AC5: Integrado en kokoro-gads.md session_log
